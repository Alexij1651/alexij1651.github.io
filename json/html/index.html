JavaScript · Actividades 1 – 7 · Interactivos

ACT 1 JSON.parse + manejo de errores 
// JSON válido
const texto = '{"nombre":"Ana","edad":30}';
 
try {
  const obj = JSON.parse(texto);
  console.log(obj.nombre);  // "Ana"
} catch (e) {
  console.error("JSON inválido:", e.message);
}
▶ Salida
// JSON inválido – prueba de error
const malo = "{ nombre: 'Ana' }";
try {
  JSON.parse(malo);
} catch (e) {
  console.error("Error:", e.message);
}
▶ Error capturado
ACT 2 JSON.stringify formateado 
const usuario = { nombre: "Pedro", edad: 40 };
 
// Convertir a JSON formateado (2 espacios)
const json = JSON.stringify(usuario, null, 2);
console.log(json);
▶ Resultado
ACT 3 Corrección de JSON inválido
❌ Código con errores:

const mal = "{ nombre: 'Ana' }";
JSON.parse(mal);
Errores encontrados:
1 Las claves deben ir entre comillas dobles: "nombre"
2 Los valores string deben usar comillas dobles: "Ana"
3 JSON no admite comillas simples (solo dobles)
✅ Código corregido:

const bien = '{"nombre": "Ana"}';
const obj = JSON.parse(bien);
▶ Resultado correcto
ACT 4 Array de personas · filtro y media 
const personasJSON = `[
  {"nombre":"Ana","edad":30},
  {"nombre":"Pedro","edad":40},
  {"nombre":"Lucia","edad":25}
]`;
 
const personas = JSON.parse(personasJSON);
 
// Mayores de 30
const mayores = personas.filter(p => p.edad > 30);
